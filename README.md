# DevDock – Open Source Developer Toolkit

A browser-based developer toolkit with JSON Formatter, Markdown Previewer, Commit Message Generator, and Snippet Saver.

## Tech Stack

- React (Vite)
- TailwindCSS
- LocalStorage
- react-markdown

## Setup

1. Install dependencies

```bash
npm install
```

2. Start the dev server

```bash
npm run dev
```

Open the Local URL shown in the terminal.

The app is frontend-only and works offline (localStorage-based). It can be deployed to Netlify or Vercel without additional configuration.
